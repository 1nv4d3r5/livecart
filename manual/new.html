Languages

Manage Languages

LiveCart supports multiple languages for the same system, which enables you to adjust your shop for international usage. You can translate both interface information (menus, captions, instructions, etc.), as well as most of the data, which is stored in the system, like product descriptions, category names and others. This may seem overhelming at first but as you will see there a couple of handy tools to help you deal with interface translations (see Live Translation).

Things you can DO

  • To add a new language
  • Modify translations
  • Sort / Reorder languages (impact on "Chance Language" menu
  • Enable or disable a language
  • Set default language (interface lang. diference)
  • Delete language

Adding a language is a one step task. Simply select language from the list and click OK. Added languages appear below automatically.

To enter translations proceed to the Language translation page (link here?).

To reorder languages click on any language and drag it around to set the preferred arrangement. Language arrangement affects the order in which languages are displayed in language switching menus.

The newly added languages are inactive by default. You might want to keep a language inactive while you are still translating the interface or product data. To activate a language mark its check_box on the left of the flag icon. To deactivate a language clear a check_box by the language you want to deactivate.

To delete a language, hover over it and click the delete icon on the left

To change default language... consequences...

To apply new language to the interface, click Change Language in the LC's main window and choose a language from the list dipslayed. Note: if the language is not fully translated it will use translations from the default language to fill empty translations (see setting Default Language)


Editing Interface Translations

Before editing translations you can have them filtered by their current status in terms of translation process. You can select the following filtering options:

  • All – show all the existing words
  • Translated – display only translated items
  • Not translated – show items that have not been translated
