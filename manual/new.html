Languages


Manage Languages

LC allows/enables you to adjust your shop for international usage, to prepare translations for other languages you have to set the interface (objects') translations and all the available stock information (description details, for instance.). This may seem overhelming at first but as you will see there a couple of handy tools to help you deal with interface translations (see LiveTranslation)

Things you can DO

  • To add a new language
  • To set(edit) translations (filter, search, browse through tree, enter, save)
  • To apply language to the interface
  • Sort / Reorder languages (impact on "Chance Language" meniu
  • Set Active / Inactive
  • Set default language (interface lang. diference)
  • Delete language
  • Find a SPECIFIC word

Adding a language is a one step task. Simply select language from the list and click OK.

To enter translations proceed to the Language translation page (link here?).

Added languages appear below automatically. To reorder languages click on any language and drag it around to set the preferred arrangement.

All (the newly added)languages are inactive by default. You might want to keep a language inactive while you are still translating it. To activate a language mark it's check_box on the left of the flag icon. (Also) note that only active languages are available in the change iterface language meniu. To suspend a language clear a check_box by the language you want to deactivate.

To delete a language, hover over it and click the delete icon on the left

To apply new language to the interface, click Change Language in the LC's main window and choose a language from the list dipslayed. Note: if the language is not fully translated it will use translations from the dafault language to fill empty translations (see setting Default Language)


Before editing translations you can have them filtered by their current status in terms of translation process. You can select the following filtering options:

  • All – show all the existing words
  • Translated – display only translated items
  • Not translated – show items that have not been translated

You can suspend a language by clearing the check_box (unchenking the box) in front of the language. All the languages are set Active by default.
