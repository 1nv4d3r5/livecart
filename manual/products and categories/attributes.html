Attributes

Manage Attributes

Attributes is what makes it easy for a customer to find exactly he/she is looking for. For instance, if a person is looking particularly for an AMD processor and you have a category for (keeping) variuos Processors (Intel, etc.) you can create an attribute (for that category) named Processor Types with (the) values of all the available processor types. This way a customer will have an ability to select certain kind of processor without a need to browse trough all the available types(thus restricting his/hes search to specific needs). Of course you can create (and you should) as many attributes as possible (for instance, for size, brand, capacity, price and so on) which will let/allow users to customize their search in the most effective way. Note that actual filtering options are set in the Filters section where you use category's attributes to create filters.

Note: Attributes have to be set up for each category individually.
