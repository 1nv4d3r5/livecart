Manage Languages

LiveCart supports multiple languages for the same shop / system, in order to adjust your store for other languages you have to prepare translations. Translating a language includes interface elements and all the available stock information (description, details, for instance) (This may seem overhelming at first but as you will see there a couple of handy tools to help you deal with interface translations (see LiveTranslation))

LiveCart supports multiple languages for the same system, which enables you to adjust your shop for international usage. You can translate both interface information (menus, captions, instructions, etc.), as well as most of the data, which is stored in the system, like product descriptions, category names and others. This may seem overhelming at first but as you will see there a couple of handy tools to help you deal with interface translations (see Live Translation).

Things you can DO

  • Change working language Apply language to the interface
  • Add a new language
  • Set (edit) translations
  • Sort / reorder languages
  • Enable or disable languages
  • Change default language (interface lang. diference)
  • Remove language

Change working language

To change the working (interface) language, click the Change Language link in the LiveCart's main window (top-right corner) and choose a language from the list displayed.

Note: main language of the interface holds definitions of all the elements (menius, buttons, etc) while the default language defines stock info (details, description, etc.). Thus changing Interface language and Default language impacts interface diferently

Add new language

Adding a language is a one step task. Simply select language from the list and click ADD. Added languages appear below automatically.

To add a language click on Add New Language below Manage Languages heading and select a languages, followed by the Add button.

Enable or disable languages

All the newly added languages are inactive by default. You might want to keep a language inactive while you are still translating it. To activate a language mark it's check_box on the left of the flag icon. (Also) note that only active languages are available in the change iterface language meniu. To suspend a language clear a check_box by the language you want to deactivate.

(a screenshot would be nice)

Also note that the disabled languages will still be active in backend, so you could prepare all the necessary translations before displaying any content in these languages before the translations are completed. When you're done with the translations just activate the language to make it available in site frontend as well.

Change default language

Defaut language in LC is used to separate stock info from the interface language (with the purpose ...) thus the default language holds your shop's stock info such as category details, descriptions, etc.

The default language serves two main purposes:

  • It is the default language your site visitors see when first visiting the site
  • Any translations that are missing for any other language (interface, product or category information, etc.) are automatically taken from the default language

To change the default language click the Set as default link under the language name.

Remove a language

To delete a language, hover over it and click the delete icon on the left.
