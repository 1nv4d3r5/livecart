Definitions can be set in the keyword tree below the Translation Filter. Keywords are classified by the interface layout ...

To set translations for the language you can choose "Not translated" in the Translation filter to dislpay keywords that have not been defined yet. This will cause the keyword tree to expand below. In case the tree seems too complex you might try browsing it by category branches. Simply collapse unwanted (irrelevant) result sections and start at any point you need.

You can set product info only in Products and Categories section (see Link for more info)

If you need to find a specific word or edit a definition, use a Search field in the Translation Filter area

You can have keywords filtered by their current status in terms of translation process. You can select the following filtering options:

  • All – show all the existing words
  • Translated – display only translated items
  • Not translated – show items that have not been translated

(There is no graphical representation of the keyword's object you are working on. Try to trace keyword's actual place by the category it is assigned to)
