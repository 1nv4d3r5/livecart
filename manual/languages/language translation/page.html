Translate interface elements

There are two ways of translating the interface - either by using the language translation panel or Live Translation.

Translation definitions can be set in the keyword tree below the Translation Filter. Keywords are categorized as they appear in the interface layout. (help on navigating the tree?)

You can set products info only in the Products and Categories section. All the translations are set in the apropriate Products and Categories section, for instance Category Details are set in the Category Details Tab and so on. (see Link for more info)

Find and translate a specific word or sentence

If you need to find a specific keyword or edit a definition, use a Search field in the Translation Filter area. Translations will also be searched by their value in English as well as keyword name.

Find missing translations

You can have keywords filtered by their current status in terms of translation process. You can choose from the following filtering options:

  • All – show all the existing words
  • Translated – display only translated items
  • Not translated – show items that have not been translated

Translate product info

You can translate product, category, attribute and other data directly from their management pages, for example, product modification page. The fields to enter the translations are typically placed underneath the main data form.
