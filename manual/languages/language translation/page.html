Here you can set up translations by supplying keywords with appropriate language definitions.

Definitions can be set in the keyword tree below the Translation Filter. Keywords are categorized as they appear in the interface layout. (help on navigating the tree?)

You can set products info only in the Products and Categories section. All the translations are set in the apropriate Products and Categories section, for instance Category Details are set in the Category Details Tab and so on. (see Link for more info)

If you need to find a specific keyword or edit a definition, use a Search field in the Translation Filter area

You can have keywords filtered by their current status in terms of translation process. You can choose from the following filtering options:

  • All – show all the existing words
  • Translated – display only translated items
  • Not translated – show items that have not been translated

(There is no graphical representation of the keyword's object you are working on. Try to trace keyword's actual place by the category it is assigned to)
