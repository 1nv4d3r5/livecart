Translate languages

LC can support almost all the languages of the world.

(1) To translate your shop's interface to other languages you have to translate certain phrases to their appropriate equivalents

(2) To prepare your language for usage you have to translate appropriate keywords from English to your chosen language.

Translate interface elements

There are two ways of translating the interface - either by using a keyword tree or Live Translation.

The keyword tree is placed below the Translation Filter. All the keywords in the tree are categorized as they appear in the interface layout. To supply translations for appropriate keywords browse the tree and enter corresponding translations. For example:

  • where text field is a place for translations;
  • _menu_users is a variable name;
  • Users is a default definition which is used if text area is left empty

Reference: see translating product info

Find and translate a specific word or sentence

If you need to find a specific keyword or edit a definition, use a Search field in the Translation Filter area. (Translations will also be searched by their value in English as well as keyword name).

Find missing translations

You can have keywords filtered by their current status in terms of translation process. The following filtering options are available:

  • All – show all the existing words
  • Translated – display only translated items
  • Not translated – show items that have not been translated

Translate a word or sentence directly from user interface

Live Translation is a LiveCart tool which allows you to translate a word or sentence directly from the page you see it (in). To use Live Translation, you have to turn it on the Customization Mode... (not completed yet)

Live Translation is a real-time translation tool which enables you to translate any element of the interface directly in the page you see it in. To do that you have to turn on Live Translation in the ... mode. Any interface phrase can be translated simply by clicking Live Translation once you hover over an element and a reference pops-up. We highly recommend to use Live Translation when you are not sure about the keyword's exact place in the interface.

Reference: see Translate Interface Elements

Translate product info

You can translate products, categories, attributes and other data directly from their management pages, for example, products modification page. The fields for entering translations are typically placed below the main data form.

Reference: see translating interface elements
